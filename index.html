Welcome to thecroydonproject.

This is a small project by the Go and AWS Croydon Interest group. The aim of the project is to

  1. give the group something to talk about
  2. apply new technologies
  3. teach and learn
  4. write code and deploy

what is it

The project aims to collect and provide access to crystal palace fc results since 1936. the following modules make up the project :

1) data scrapper - collects game data from various websites 2) data source - a file server to hold data scrapped from website 3) data upload - uploads data in 2 into a postgres db 4) db - holds holds game data 5) db service - restful interface to postgres data 6) front module - consumes the service in 5 and renders output

Technology

  1. Go
  2. Kubernetes
